# üìùDESCRIPTION
A FiveM script that spawns interactive NPCs with customizable menus, allowing players to access tutorials, teleportation, commands, webhooks, and other features through a contextual UI. 

# ‚ú®FEATURES
- Debug ‚Äî Debug mode: true enables debug messages, false disables them.
- Locale ‚Äî Language setting: 'en' for English, 'de' for German, 'fr' for French, 'es' for Spanish.
- Framework ‚Äî Framework selection: 'qb' for QBCore, 'qbox' for QBox Framework, or 'esx' for ESX.
- Menu ‚Äî Menu system: "ox" for ox_lib context menus, "qb" for qb-menu.
- Inventory ‚Äî Inventory system: "ox" for ox_inventory, "qb" for qb-inventory.
- Target ‚Äî Target system: "ox" for ox_target, "qb" for qb-target, or disabled.
- TextUI ‚Äî TextUI system: "ox" for ox_lib TextUI, "qb" for QBCore TextUI, or disabled.
- Notification ‚Äî Notification system: "ox" for ox_lib, "qb" for QBCore, "qbx" for QBox, "esx" for ESX, or "gta" for native GTA.
- NotificationPosition ‚Äî Notification position: "top", "top-right", "top-left", "bottom", "bottom-right", "bottom-left", "center-right".
- NotificationStyle ‚Äî Notification style: works only for "ox", default or custom CSS object.
- Locations ‚Äî Define interactive zones with coordinates, radius, custom notifications, blips, NPCs, and menus.
- Zone Notifications ‚Äî Custom notifications when entering or leaving defined zones.
- Blips ‚Äî Custom map blips for zones.
- NPCs ‚Äî Configurable NPCs in zones (model, scenario, location, interaction distance).
- Menus ‚Äî Multiple menu types per zone:
    ‚Äî Webhook: Contact admin, send messages to support, with embed options.
    ‚Äî Guide: Read guides and help texts with custom content, images, and metadata.
    ‚Äî Teleport: Choose a destination and teleport or set a waypoint.
    ‚Äî Command: Execute client commands from the menu.
    ‚Äî Export: Trigger exports/functions from the menu.
    ‚Äî Item: Receive items; can be limited to once per server restart.
- All menus are fully customizable (icon, title, description, metadata, options, events).
- Extendable ‚Äî Add more menus, guides, teleport locations, and features per zone.
